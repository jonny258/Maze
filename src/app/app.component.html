Sort an array

Instructions:

Hit reset array to generate a random array of values 100, then hit merge sort to sort through that array

Explanation:

Merge Sort: Merge sort is a divide-and-conquer algorithm that splits an array into smaller subarrays, sorts each of them recursively, and then merges them back into a single sorted array. It divides the array in half repeatedly until each piece is small enough (usually one element), at which point these pieces are inherently sorted. The sorted subarrays are then merged back together in a way that results in a fully sorted array, making merge sort particularly efficient for large datasets.

Search a Maze

Instructions:

Click the buttons to the left and see how the maze is explored and how the maze finds the shortest path

Explanation:

Breadth-first search (BFS) is an algorithm that explores a graph or tree layer by layer, starting from a selected node and traversing all its neighbors before moving to the next level of nodes. It uses a queue to keep track of nodes to visit next, ensuring that nodes are explored in the order they are discovered. Depth-first search (DFS), on the other hand, explores as far as possible along each branch before backtracking, diving deep into a graph or tree by following each path to its end before moving on to the next path. DFS typically uses a stack, either explicitly or through recursion, to keep track of the nodes to be explored.
